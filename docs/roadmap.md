  【サービス概要】
  AWS資格（CLF / SAA / SAP）の練習問題サービス。
  演習モード・模試モードを提供し、ユーザーの学習履歴・苦手傾向を分析できる。

  【技術スタック】

  レイヤー        技術
  -----------------------------------------------------------------------
  フロントエンド  React（TypeScript）
  ホスティング    AWS Amplify
  ソース管理      AWS CodeCommit
  認証            Amazon Cognito
  API             Amazon API Gateway + AWS Lambda（Node.js）
  データベース    Amazon DynamoDB
  メール送信      Amazon SES
  権限管理        AWS IAM

  【全体フェーズ】

4. Phase 3　演習モード
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  目的: ユーザーが問題を解いて正誤を確認できるコア機能を完成させる。

  【作業内容】

  #   作業                                    備考
  -----------------------------------------------------------------------
  1   POST /sessions APIの実装                セッション開始
  2   POST /sessions/:id/answers APIの実装    回答記録（3テーブル同時更新）
  3   PUT /sessions/:id APIの実装             セッション終了
  4   GET /sessions/active APIの実装          未完了セッション検出
  5   GET /sessions/:id/progress APIの実装    途中再開用
  6   演習設定画面の実装                       問題数・分野・不正解数フィルタ
  7   演習画面の実装                           1問ずつ表示・即時正誤表示・解説
  8   結果画面の実装（演習用）                 スコア・合否・正誤一覧
  9   ログイン時の途中再開ダイアログ実装

  【回答記録時の3テーブル同時更新（Lambdaトランザクション）】

  UserAnswers        ← 生データ追記
  UserQuestionStats  ← correctCount or incorrectCount をインクリメント
  UserTagStats       ← 該当タグの正答率を更新

  【演習モードの仕様】

  項目            内容
  -----------------------------------------------------------------------
  問題数          ユーザーが任意に指定
  時間制限        なし
  正誤表示        1問解答ごとに即時表示
  解説表示        正誤と同時に表示
  合否判定        正答率65%以上で合格
  途中中断        対応（次回ログイン時に再開可能）

  【途中再開フロー】

  ログイン
    └→ GET /sessions/active
          ├→ 未完了あり → 「再開しますか？」ダイアログ
          │      ├→ 再開 → GET /sessions/:id/progress → 途中から復元
          │      └→ 破棄 → PUT /sessions/:id（abandoned） → ホームへ
          └→ なし → ホームへ

  【Phase 3 完了条件】

  - 演習設定画面で問題数・フィルタを指定して開始できる
  - 1問ずつ解答し、即時に正誤と解説が表示される
  - 結果画面でスコアと合否が表示される
  - 途中でブラウザを閉じても次回ログイン時に再開できる

  【模試モードの仕様】

  資格    問題数  制限時間  合否ライン
  -----------------------------------------------------------------------
  CLF     65問    90分      正答率65%以上
  SAA     65問    130分     正答率65%以上
  SAP     75問    180分     正答率65%以上

  【公式出題比率（模試での問題配分に使用）】

  CLF:
    クラウドのコンセプト              24%（約16問）
    セキュリティとコンプライアンス    30%（約20問）
    クラウドテクノロジーとサービス    34%（約22問）
    請求・料金・サポート              12%（約8問）

  SAA:
    セキュアなアーキテクチャの設計          30%（約20問）
    弾力性に優れたアーキテクチャの設計      26%（約17問）
    高パフォーマンスなアーキテクチャの設計  24%（約16問）
    コスト最適化されたアーキテクチャの設計  20%（約13問）

  SAP:
    組織の複雑さに対応したソリューションの設計  26%（約20問）
    新しいソリューションの設計                  29%（約22問）
    既存ソリューションの継続的改善              25%（約19問）
    ワークロードの移行とモダナイゼーション      20%（約15問）

10. データモデル一覧
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  【Questions】
  PK: questionId
  属性: examType, questionText, choices[], correctAnswers[],
        explanation, tags[], isMultiple, createdAt
  GSI: examType-index（examTypeをPKとして問題一覧を取得）

  【Tags】
  PK: tagId
  属性: name, examType, officialRatio

  【QuestionTagRelations】
  PK: tagId / SK: questionId
  属性: examType

  【UserAnswers】
  PK: userId / SK: questionId#timestamp
  属性: isCorrect, selectedAnswers[], sessionId, answeredAt

  【UserQuestionStats】
  PK: userId / SK: questionId
  属性: correctCount, incorrectCount, lastAnsweredAt

  【UserTagStats】
  PK: userId / SK: tagId
  属性: correctCount, incorrectCount

  【Sessions】
  PK: userId / SK: sessionId
  属性: mode, examType, questionIds[], status,
        startedAt, endedAt, score, isPassed, lastAnsweredAt
  status の値: active / completed / abandoned

  【Reports】
  PK: questionId / SK: reportId
  属性: userId, message, reportedAt
